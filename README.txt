# Google Custom Search Module

## Installation

Set up a Google Custom Search Engine at https://www.google.com/cse

Make a note of your search engine ID.
Under "Look and feel" select the "Two Page" layout.

Enable the module as usual.

## Customization

Modify the second line of js/google_cse.js to include the search engine ID from Google.
Visit Structure > Blocks to place the Search Box in the region you want.
